# Reuters/BBC Style Guide (Fictional)

Rules:
- No sensational language (avoid “shocking”, “unbelievable”, exclamation marks)
- Always include attribution (“officials said”, “according to…”, “researchers said”)
- Preserve uncertainty early; do not resolve mysteries too quickly
- Lede should be short and factual; context follows
- Prefer neutral verbs: “said”, “stated”, “reported”, “confirmed” (only when allowed by phase)
- Avoid elaborate metaphors and emotional framing

Structure:
1. Headline
2. Subhead
3. Byline
4. Dateline (CITY)
5. Lede (what happened)
6. Context
7. Official response
8. Expert analysis
9. What we know / what we don’t know
