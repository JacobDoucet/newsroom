# Editorial Rubric

Scores (0–5):
- Journalism tone (restrained, neutral)
- Attribution (claims have sources)
- Uncertainty preserved (not over-confirming)
- Pacing (fits phase; no sudden jumps)
- Coherence (references known facts correctly)
- Clarity (clean structure, readable lede)
- Intrigue (compels without sensationalism)

Default review tags:
- too_dramatic
- breaks_canon
- too_fast
- not_reuters
- unclear_lede
- weak_headline
- too_many_claims
- boring
